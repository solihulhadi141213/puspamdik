# PUSPAMDIK - Website Profil Lembaga

**PUSPAMDIK** (Pusat Studi Pengembangan Mutu Pendidikan) adalah platform profil lembaga yang berfungsi sebagai pusat informasi, layanan, dan standar pengembangan mutu pendidikan. Website ini dirancang untuk memberikan informasi yang transparan, profesional, dan mudah diakses oleh publik.

---

## 🛠️ Teknologi & Stack

Website ini dikembangkan dengan teknologi web standar industri untuk memastikan performa yang cepat dan antarmuka yang modern:

* **Core:** PHP, HTML5
* **Logic & Interactivity:** jQuery & JavaScript
* **Styling:** CSS3 & Bootstrap 5

## 📦 Dependensi

Proyek ini menggunakan beberapa library pihak ketiga untuk meningkatkan fungsionalitas dan pengalaman pengguna:

| Library | Versi | Deskripsi |
| :--- | :--- | :--- |
| **Bootstrap** | `^5.3.8` | Framework CSS untuk tata letak responsif dan komponen UI. |
| **Bootstrap Icons** | `^1.13.1` | Pustaka ikon resmi dari Bootstrap. |
| **AOS (Animate On Scroll)** | `^2.3.4` | Menambahkan efek animasi saat elemen masuk ke viewport. |
| **Swiper** | `^12.1.3` | Slider modern untuk galeri, berita, atau testimoni. |
| **PureCounter.js** | `^1.5.0` | Animasi penghitung angka untuk statistik lembaga. |
| **Validate.js** | `^2.0.1` | Library untuk validasi input pada formulir kontak atau registrasi. |

---

## 🚀 Fitur Utama

1.  **Fully Responsive:** Tampilan website menyesuaikan secara otomatis baik di perangkat mobile, tablet, maupun desktop.
2.  **Dynamic Content:** Menggunakan PHP untuk manajemen konten yang lebih dinamis.
3.  **Modern UI/UX:** Animasi yang halus menggunakan AOS dan slider interaktif dari Swiper.
4.  **Client-side Validation:** Validasi formulir yang cepat dan user-friendly sebelum data dikirim ke server.

## 💻 Cara Instalasi

1.  **Clone atau Download**
    Simpan folder proyek di direktori server lokal Anda (misal: `htdocs` untuk XAMPP).
2.  **Konfigurasi Server**
    Pastikan server lokal Anda mendukung PHP versi 8.0 atau yang lebih baru.
3.  **Instalasi Dependensi**
    Jika menggunakan npm, jalankan:
    ```bash
    npm install
    ```
    Atau pastikan file aset di folder `vendor` sudah terhubung dengan benar di file HTML/PHP.
4.  **Akses Website**
    Buka browser dan akses:
    ```
    http://localhost/puspamdik
    ```

## 📁 Struktur Folder (Opsional)
* `assets/` - Folder untuk CSS, JS, dan Gambar kustom.
* `vendor/` - Folder untuk library/dependensi pihak ketiga.
* `includes/` - Kumpulan file PHP (header, footer, config).
* `index.php` - Halaman utama.
